# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Giving and receiving feedback professionally
- Focusing on what is best for the project and community

Examples of unacceptable behavior:

- Harassment, insults, or discrimination
- Trolling, personal attacks, or inflammatory comments
- Publishing private information without explicit permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers via GitHub issues or private channels.

## Scope

This Code of Conduct applies within project spaces and public spaces when an individual is representing the project.
