# Security Policy

## Supported Versions

Security fixes are prioritized for the latest release.

## Reporting a Vulnerability

Please report security issues privately and do not open public issues for vulnerabilities.

You can report by opening a private security advisory on GitHub:

- Go to `Security` tab in this repository
- Click `Report a vulnerability`

If GitHub Security Advisories are not available, contact maintainers directly.

## Response Process

- We will acknowledge receipt as soon as possible.
- We will investigate and validate the issue.
- We will coordinate a fix and release.
- We will provide disclosure details after a patch is available.
